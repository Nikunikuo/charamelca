キャラメルカ！ version 1.0.0

このアプリは、オリジナルキャラクター設定＋キャラクター同士のストーリー生成ができるツールです。
ブラウザ版として作成したものをElectronでデスクトップアプリ化しています。

【使い方】
起動は簡単！キャラメルカ！のexeをダブルクリックです。

1. 左側でキャラクター設定を入力し、「キャラクター生成」ボタンを押してください。
下側の欄に生成されたプロンプトがポン出しされます。
2. 出力されたプロンプトを一番下の「送信」ボタンでAPIに飛ばします。
プロンプトの中身は直接いじってもらっても構いません。
左側のキャラクター設定での空欄は適当にAIが補完します。
3. 出力されたキャラクター設定は、保存ボタンでフォルダを指定して保存できます。
4. 保存したキャラクターたちをフォルダごと読み込むこともできます。（画像同梱対応）
5. キャラクターたちの関係性を自由にストーリー化して楽しめます。
　 キャラクターを読み込んだ後、複数のキャラクターにチェックを入れ、生成ボタンを押すと
　 そのキャラクター達が登場するストーリーを生成します。また、シチュエーション入力などにも対応していますので
　 満足のいくまで遊び倒してください。

【フォルダ読み込みについて】
- .txtファイル（キャラ設定）と、同名の.pngファイル（キャラ画像）をセットで管理してください。
- 例）ミラ.txt ＋ ミラ.png

【注意事項】
- このアプリはインターネット接続とOpenAI APIキーが必要です。
- APIキーは自己責任で管理してください。
- APIを呼び出して使うため、当然APIのトークンは消費します。使いすぎにご注意下さい。
